The final delivery required the design, implementation and usability inspection of a website for a fictitious no-profit organization, an Italian association that operates in order to promote day-to-day supportive activities and beautiful events inside hospitals. The project has been graded 30/30 in the Hypermedia Applications course.

Tools used:

  • Interactive Dialogue Models
  • ER diagram and Relational Tables
  • Lo-Fi and Hi-Fi Design with Sketch

Frameworks used:

  • Front-end with pure HTML, CSS
  • jQuery, Bootstrap
  • Backend with NodeJs
  • REST APIs
  • SQL database
