KINGARTOUR

Apple Developer Academy
June 2018

Challenge

App

How can we make art education more engaging and immersive? How come people are not really interested in art? KINGARTOUR is an application developer for iPad that allow the user to visit virtual museums using AR with certified informations about artists and artwork. It is designed for education, teachers could use it in classrooms.

Tools used:

  • Challenge-Based Learning approach
  • ER diagram and Relational Tables
  • Value Proposition
  • Competitors Analysis

Frameworks used:

  • Lo-Fi and Hi-Fi Design with Sketch
  • Visual Design with Ai, Ps, Blender
  • Front-end with Swift (ARKit)
  • Backend with Laravel, phpMyAdmin
  • Github
  • SQL database
